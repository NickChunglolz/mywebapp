The previous version of this site was a `vue-cli` scaffold from 2017 with three files in `src/`. It loaded a router and nothing else. That felt about right for how often I update personal sites, but it didn't represent me anymore.

So I tore it down and rebuilt on the stack I actually use for side projects today: **Next.js 16, React 19, Tailwind v4, TypeScript**. Same stack as [Drip](https://github.com/NickChunglolz), my Canadian subscription tracker. Fewer decisions to re-make is the whole point.

## What's here now

- **About / Work / Projects / Skills / Contact** — the standard sections, but written like I'd actually describe myself, not like a recruiter would.
- **Blog** — short technical posts. The plan is to write something every time I learn a thing worth remembering. If you're reading this, you're already in it.

## Why the dark theme and monospace bits

I look at terminals all day. Sites with that visual language feel like home. The accent gradient (indigo → cyan) is a nod to Caper's brand palette — small Easter egg for anyone from the team.

- Probably a `/uses` page eventually. Probably.
